# Archive — superseded implementations

Everything in this folder is inert and kept only for reference. The single live
implementation of Amy's Bydgoszcz Treasure Trail is the React + TypeScript +
Vite app in `/app`, deployed to Vercel (with a built mirror committed at the
repo root for GitHub Pages).

- `index.html` + `standalone/app.js|app.css` — the hand-written vanilla EN
  version (route passport + memory keepsake). Its 10-stop content and memory
  feature were ported into the React app.
- `index-pl.html` + `standalone/app-pl.js` — the Polish twin of the above.
- `site/` — an old built Vite bundle previously mirrored for GitHub Pages.
- `city-screenshot.png` — a stray full-page screenshot (was the root `City` file).
